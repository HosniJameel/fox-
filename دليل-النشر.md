## النشر على GitHub Pages (مجاني ودائم)

**١. سوّي حساب** على github.com (لو ما عندك).

**٢. سوّي Repository جديد:**
- اضغط **New** → اسم مثلاً `fox-prizes` → خليه **Public** → **Create repository**.

**٣. ارفع الملفات:**
- افتح الـ repo → **Add file** → **Upload files**.
- اسحب **محتوى فولدر `site`** (كل ملفات `.html`) وارميهن.
- تحت، اضغط **Commit changes**.

**٤. فعّل الصفحات:**
- **Settings** (فوق) → **Pages** (بالجانب).
- تحت **Source** اختر **Deploy from a branch**.
- **Branch:** `main` و **Folder:** `/ (root)** → **Save**.
- انتظر دقيقة، راح يطلعلك رابط أخضر فوق.

**٥. روابط جوائزك** تصير بهذا الشكل (بدّل `USERNAME` و `fox-prizes` بمالك):
```
https://USERNAME.github.io/fox-prizes/playstation.html
https://USERNAME.github.io/fox-prizes/pc.html
https://USERNAME.github.io/fox-prizes/billiard.html
https://USERNAME.github.io/fox-prizes/snooker.html
https://USERNAME.github.io/fox-prizes/vip.html
https://USERNAME.github.io/fox-prizes/restaurant.html
https://USERNAME.github.io/fox-prizes/bar.html
```

**٦. سوّي الباركود** من أي موقع باركود مجاني وحط رابط كل جائزة.

> نصيحة: أي تعديل مستقبلي، بدّل الملف بنفس الـ repo (Add file → Upload files) ويتحدّث الموقع تلقائياً خلال دقيقة.

## تعديل الجوائز أو الموقع لاحقاً

كل التعديلات بمكان واحد داخل أي ملف، بأول قسم اسمه **الإعدادات**:

- **موقع الخريطة:** بدّل قيمة `mapsUrl`.
- **مدة الصلاحية:** غيّر `expiryDays`.
- **نصوص الجوائز:** عدّل الاسم/الوصف/الأيقونة داخل `prizes`.

لو تغيّر جائزة، عدّلها بملفها المستقل + بـ `index.html`.

## شنو يصير للزبون

- يقرأ الباركود → تنفتح صفحة **ألف مبروك** بجائزته.
- الجائزة **تنحفظ تلقائياً بجهازه**، فكل ما يرجع يفتح الرابط تظل تطلعله كتذكير.
- زر **الموقع** يفتحله المحل على الخرائط مباشرة.
- زر **ذكّرني** ينزّل تذكير بتقويم جواله.
- كل زبون عنده **كود خاص** (مثل `FOX-3-K9P2`) يعرضه عند الكاونتر — يمنع التكرار.
- بعد الاستلام يضغط **علّمها مستخدمة** فتوقف التذكيرات.

> ملاحظة: **الإشعار التلقائي عند الاقتراب من المحل** ما ينفّذ بصفحة ويب (خصوصاً آيفون) — يحتاج تطبيق فعلي. البديل المطبّق: الحفظ بالجهاز + تذكير التقويم، ويشتغل على كل الجوالات.
