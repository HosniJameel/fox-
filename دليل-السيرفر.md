# تشغيل نظام الأكواد (Supabase) — ١٠٠ كود لكل جائزة 🦊

الهدف: كل رابط بيه ١٠٠ كود فريد · كل زبون ياخذ كود جديد · يتوقف بعد ١٠٠ · كل كود ينستخدم مرة وحدة بتأكيد الموظف.

> ملاحظة: قبل ما تعبّي بيانات السيرفر، الصفحات تشتغل **بوضع محلي** (كل جهاز ياخذ كود)، بس بدون سقف ١٠٠ وبدون متابعة مركزية. بعد ما تكمل الخطوات تحت، يصير النظام حقيقي.

---

## ١) سوّي مشروع Supabase (مجاني)

1. ادخل **supabase.com** → **Start your project** → سجّل دخول (بحساب GitHub أسهل).
2. **New project** → اختر اسم `fox` → حط **Database Password** (خزنها) → اختر أقرب Region → **Create**.
3. انتظر ~دقيقتين لحد ما يجهز المشروع.

## ٢) سوّي الجداول والدوال

1. من القائمة اليسار: **SQL Editor** → **New query**.
2. افتح ملف **`supabase/1_schema.sql`** وانسخ محتواه كامل والصقه → اضغط **Run**.
   - 🔐 قبل ما تضغط Run، بدّل `'1234'` بالرقم السري (PIN) الي تريده للموظفين (خليه أطول لأمان أكثر).

## ٣) عبّي الأكواد (٧٠٠ كود)

1. **New query** مرة ثانية.
2. افتح **`supabase/2_seed_codes.sql`** والصق محتواه كامل → **Run**.
3. للتأكد: **Table Editor** → جدول `prize_codes` لازم يصير فيه ٧٠٠ صف.

## ٤) خذ رابط ومفتاح المشروع

- من اليسار: **Project Settings** (الترس) → **API**.
- انسخ:
  - **Project URL** (مثال: `https://abcd1234.supabase.co`)
  - **anon public** key (المفتاح الطويل تحت Project API keys)

## ٥) ركّبهن بالصفحات

عندك خياران:

**الأسهل:** دزلي الـ URL والـ anon key، وأنا أعبّيهن بكل الصفحات الثمانية وأرجّعهن لك جاهزين.

**أو بنفسك:** بكل ملف `.html` (وبـ `index.html`)، بأول قسم **الإعدادات**، بدّل:
```js
supabaseUrl: "",   // حط Project URL هنا
supabaseKey: "",   // حط anon public key هنا
```
بعدها ارفع الملفات المحدّثة للـ GitHub مرة ثانية.

## كيف تراقب وتتحكم (من SQL Editor)

**كم باقي بكل جائزة:**
```sql
select prize_id, status, count(*)
from prize_codes group by prize_id, status order by prize_id, status;
```

**تصفير جائزة معيّنة (ترجع كل أكوادها متاحة):**
```sql
update prize_codes set status='free', device_id=null, claimed_at=null, redeemed_at=null
where prize_id='1';   -- 1=بلايستيشن ... 7=البار
```

**إضافة أكواد أكثر لجائزة (مثال ٥٠ كود للجائزة 1):**
```sql
insert into prize_codes (code, prize_id)
select 'FOX-1-'||upper(substr(md5(random()::text),1,5)), '1'
from generate_series(1,50)
on conflict (code) do nothing;
```

**تغيير الـ PIN لاحقاً:** افتح `1_schema.sql`، بدّل `'1234'`، وأعد تشغيل جزء دالة `redeem_code` فقط.

## ملاحظات أمان بسيطة
- مفتاح `anon` عادي يكون بالصفحة — آمن لأن الجدول مقفول (RLS) وما ينقرأ إلا عبر الدوال.
- خلّي الـ PIN مو سهل (٦ أرقام أحسن) لأن التأكيد يصير من الصفحة.
- خريطة الجوائز: 1=بلايستيشن · 2=PC · 3=بيليارد · 4=سنوكر · 5=VIP · 6=المطعم · 7=البار · (حظ أوفر بلا أكواد).
